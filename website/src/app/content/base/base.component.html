Slap Countdown

Slap Countdown is a simple site where you can create a slap-countdown like the slap bet from "How I met your mother".

On the right you can create a simple countdown and a link will be provided which you can share to your slap buddy.

Create Countdown

Below you can create a slap countdown.

Sender is required. Sender name is too long (max 20 characters).
Receiver is required. Receiver name is too long (max 20 characters).
Date & time are required.
Reason is required. Reason name is too long (max 255 characters).
